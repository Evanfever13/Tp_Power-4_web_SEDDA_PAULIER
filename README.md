# Tp_Power-4_web_SEDDA_PAULIER

## Description
Ce dépôt contient le projet "Tp Power 4" réalisé dans le cadre du cours web (HTML/CSS). Il s'agit d'une petite application/site statique démontrant la mise en page, le responsive design et quelques interactions simples côté client.

## Installation et exécution
- Golang 1.25.0
- Utiliser la commande `go run .` dans le dossier `src` dans votre IDE

## Utilisation
- Naviguer via le fichier `home.html`.

## Structure du projet
- assets (Image/Css/Police)
- templates (Html)
- src (Golang)
